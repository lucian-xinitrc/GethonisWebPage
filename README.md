## 2. Overview / Introduction
Gethonis is a lightweight web chatbot designed to provide quick and meaningful interactions. It can answer questions, assist with tasks, and serve as a friendly virtual assistant for your website visitors.  

**Designed for:**  
* Users seeking instant help or guidance online  
* Developers testing AI chatbot integrations in Next.js  
* Communities or projects needing a simple, deployable chatbot demo  

**Key strengths:**  
* Minimal and clean UI focused on conversation  
* Fast and SEO-friendly with **Next.js**  
* Easy to extend with AI models or custom APIs  
* Lightweight and deployable on **Vercel** in seconds*

## 3. Features
* Interactive chatbot interface for real-time conversations  
* Supports text-based queries and responses  
* Responsive design for desktop and mobile  
* Built with Next.js for speed and SEO optimization  
* Easy to integrate with AI models or other APIs in the future
